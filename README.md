# Nebula

Color picker + pixel monitor. Dark theme, lightweight, single executable.

## Download
Download `nebula` from [Releases](https://github.com/archbee/nebula/releases).

## Usage
```bash
chmod +x nebula
./nebula
```

- **Color Picker**: Move mouse, see RGB/hex
- **Pixel Monitor**: Pick a pixel, set a key — when white disappears, it presses that key
